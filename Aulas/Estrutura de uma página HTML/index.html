Estrutura de uma página HTML

De nada adianta você sair colocando tags, textos, etc se você não sabe como estruturar o seu HTML, lembre-se que depois do HTML vem o CSS. Para que o CSS possa deixar a sua página bonita, ela precisa estar bem estruturada

As tags (div) e (span) são também chamadas de tags coringas. Sempre usamos essas duas tags para estruturar uma página web, porque elas não carregam "peso semântico", então elas servem para delimitar um espaço, para depois manipularmos esses espaços com o CSS.

As (div) servem para separar um mais espaços e dizer o que erá colocado dentro delas. As div não tem atributos. A div é uma tag do tipo display block, ou seja, ela ocupa a horizontal inteira de onde ela está inserida

As tags (span) não ocupam o espaço todo de ponde está inserida, sendo assim podemos usá-las para posteriormente no CSS trablarmos somente aquele ponto. No CSS não conseguimos dar cor a uma só letra, com a tag span abraçando aquela letra ou trecho, aí conseguimos.
